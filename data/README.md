# Data Directory

This directory contains data files used by the Tour Guide system.

## Structure

```
data/
├── README.md           # This file
├── sample_routes/      # Sample route data for testing
└── cached_responses/   # Cached API responses (gitignored)
```

## Sample Data

Sample route data can be placed here for testing without API calls.

## Notes

- Large data files should be added to `.gitignore`
- Cached API responses are not committed to version control
- Sample data should be anonymized if containing real location data
