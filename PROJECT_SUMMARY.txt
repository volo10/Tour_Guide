================================================================================
                         AGENT DESCRIPTIONS
================================================================================

AGENT 1: VIDEO FINDER üé¨
Uses 4 Skills:
  1. Location Contextual Analysis - Extract geographic context
  2. YouTube Video Discovery - Find & rank videos
  3. Relevance Scoring - Score video relevance
  4. API Web Fetching - Fetch YouTube data

Scoring (100 points):
  - Relevance: 40 (location match)
  - Quality: 30 (production, authority)
  - Engagement: 20 (views, likes, recency)
  - Usefulness: 10 (practical info)

Output: YouTube video with direct link, views, and rating

---

AGENT 2: MUSIC FINDER üéµ
Uses 4 Skills:
  1. Location Cultural Mapping - Map to music cultures
  2. Spotify Track Discovery - Find & rank tracks
  3. Mood Context Matching - Match mood to location
  4. API Web Fetching - Fetch Spotify data

Scoring (100 points):
  - Relevance: 40 (location essence capture)
  - Quality: 25 (artist reputation, production)
  - Atmosphere: 20 (mood match to location)
  - Discoverability: 15 (streaming availability)

Output: Spotify track/playlist with link and streaming info

---

AGENT 3: HISTORY FINDER üìñ
Uses 4 Skills:
  1. Historical Research Analysis - Research history
  2. Fact Finding Verification - Verify facts
  3. Narrative Story Crafting - Create narratives
  4. API Web Fetching - Fetch historical data

Scoring (100 points):
  - Accuracy: 35 (fact verification, sources)
  - Engagement: 30 (story quality, fun facts)
  - Depth: 20 (comprehensive information)
  - Presentation: 15 (organization, clarity)

Output: Historical narrative with timeline and fun facts

---

AGENT 4: JUDGE üèÜ
Uses 4 Skills:
  1. Content Evaluation - Assess quality
  2. Scoring Methodology - Apply scoring
  3. Comparative Analysis - Compare recommendations
  4. Result Compilation - Generate report

Role: Evaluate all three recommendations fairly and declare winner

Output: Comprehensive evaluation report with rankings and winner declaration

================================================================================
                         SYSTEM WORKFLOW
================================================================================

INPUT: Google Maps Address
  Example: "Eiffel Tower, Paris, France"

PROCESSING (Parallel):
  1. Video Finder searches YouTube
  2. Music Finder searches Spotify
  3. History Finder researches history

EVALUATION:
  Judge Agent evaluates all three recommendations

OUTPUT: Comprehensive Report
  - Video recommendation with score
  - Music recommendation with score
  - History recommendation with score
  - Judge's evaluation and winner declaration

TIME: 10-15 seconds for full roundtrip
ACCURACY: 90%+ for well-known locations
